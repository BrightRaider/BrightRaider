[ ENGLISH ]

WHAT IS BRIGHTRAIDER?
  BrightRaider is a lightweight tool for NVIDIA and AMD graphics
  cards that lets you instantly switch between display profiles
  using your Numpad keys. Each profile changes gamma, contrast,
  and digital vibrance / saturation - perfect for gaming
  (especially Arc Raiders!), video editing, or quickly boosting
  visibility in dark scenes.

  The tool auto-detects your GPU (NVIDIA or AMD) and uses the
  correct driver API automatically. On systems without a
  supported GPU, gamma and contrast still work via Windows API.

  The app runs silently in your system tray and uses almost
  no resources. No installation required.

FREE FEATURES
  - 3 switchable display profiles via Numpad 1/2/3
  - Adjusts Gamma, Contrast, and Digital Vibrance/Saturation
  - Supports NVIDIA (NvAPI) and AMD (ADL SDK) graphics cards
  - Auto-detects GPU - no configuration needed
  - Works with Shift held down (no conflicts)
  - Per-monitor control (choose one or all monitors)
  - Notifications can be turned on/off
  - English and German language support
  - Custom icon with profile number indicator
  - All settings saved automatically
  - Resets everything to normal on exit
  - Portable: no installation, just one EXE + icon file

PRO FEATURES ($4.99 one-time)
  - Up to 9 profiles (Numpad 1-9)
  - Custom profile editor: set your own Gamma, Contrast
    and Vibrance values for each profile
  - Auto-Start with Windows: launches on boot
  - Game Mute toggle (Numpad 0): instantly mute/unmute
    game audio
  - Auto-Brightness: screen analysis detects dark scenes
    and automatically switches to a brighter profile
  - Offline license: no internet required after activation

PROFILES (Default)
  Num 1:  Normal    - Gamma 1.0, Contrast 100%, Vibrance 50%

SYSTEM REQUIREMENTS

  - Also works on Intel/other GPUs (Gamma + Contrast only)

FIRST LAUNCH - IMPORTANT
  On first launch, BrightRaider needs to make a small, safe
  change to your Windows registry. Here is exactly what
  happens and why:

  1. You double-click BrightRaider.exe
  2. Windows shows a UAC prompt ("Do you want to allow
     this app to make changes?") - click YES
  3. This creates ONE registry entry:

     Location: HKLM\SOFTWARE\Microsoft\Windows NT\
               CurrentVersion\ICM
     Name:     GdiIcmGammaRange
     Value:    256

     WHAT DOES THIS DO?
     Windows has a built-in restriction that blocks apps
     from adjusting display gamma. This registry entry
     simply tells Windows: "Allow gamma adjustments."
     It does NOT change any display settings itself.
     It does NOT affect system stability or security.
     It is used by many display calibration tools.
     It can be safely removed at any time.

  4. RESTART YOUR PC (only needed once!)
  5. Start BrightRaider.exe again - done!

  The UAC prompt only appears ONCE. After that, the app
  runs without any admin rights.

AUTO-BRIGHTNESS (Pro)
  When enabled, BrightRaider captures the center 30% of your
  screen every 3 seconds and calculates the average brightness.
  If the screen is dark (below threshold), it automatically
  switches to a brighter profile. When the screen brightens
  again, it switches back. Uses hysteresis to prevent
  flickering between profiles.

  Enable via: Tray Menu -> Auto-Brightness (Screen Analysis)

MONITOR SELECTION
  By default, all monitors change together. To change only
  one monitor: right-click the tray icon and select the
  specific display. Your choice is saved automatically.

NOTES
  - Numpad 1-9 are captured while the tool is running.
    Regular number keys (above letters) are NOT affected.
  - Digital Vibrance (NVIDIA) / Saturation (AMD) changes are
    visible in the respective GPU control panel.
    Gamma/Contrast changes are NOT shown there
    (this is normal - they work via a different API).
  - The tray menu shows your detected GPU at the bottom
    (NVIDIA, AMD, or None).
  - If something looks wrong: just exit the app. Everything
    resets to normal automatically.

UNINSTALL
  1. Exit BrightRaider (right-click tray -> Exit)
  2. Delete the folder
  3. Optional: Remove the registry entry:
     HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\ICM
     -> Delete "GdiIcmGammaRange"
  4. Optional: Remove auto-start entry:
     HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
     -> Delete "BrightRaider"

ERSTER START - WICHTIG
  Beim ersten Start muss BrightRaider eine kleine, sichere
  Aenderung in der Windows-Registry vornehmen. Hier ist genau
  erklaert was passiert und warum:

  1. Du doppelklickst BrightRaider.exe
  2. Windows zeigt eine UAC-Abfrage ("Moechten Sie zulassen,
     dass diese App Aenderungen vornimmt?") - klicke JA
  3. Es wird EIN Registry-Eintrag erstellt:

     Ort:   HKLM\SOFTWARE\Microsoft\Windows NT\
            CurrentVersion\ICM
     Name:  GdiIcmGammaRange
     Wert:  256

     WAS MACHT DIESER EINTRAG?
     Windows hat eine eingebaute Beschraenkung, die Apps
     daran hindert, die Display-Gamma-Werte zu aendern.
     Dieser Registry-Eintrag sagt Windows einfach:
     "Erlaube Gamma-Anpassungen."
     Er aendert KEINE Bildschirmeinstellungen selbst.
     Er beeinflusst NICHT die Systemstabilitaet.
     Er ist KEIN Sicherheitsrisiko.
     Er wird von vielen Kalibrierungs-Tools verwendet.
     Er kann jederzeit wieder entfernt werden.

  4. PC NEU STARTEN (nur einmalig noetig!)
  5. BrightRaider.exe erneut starten - fertig!

  Die UAC-Abfrage kommt nur EINMAL. Danach laeuft die App
  ohne Admin-Rechte.
