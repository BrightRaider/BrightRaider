================================================================
  BrightRaider
  Switch Gamma, Contrast & Digital Vibrance with one keypress
================================================================


[ ENGLISH ]

WHAT IS BRIGHTRAIDER?
  BrightRaider is a lightweight tool for NVIDIA and AMD graphics
  cards that lets you instantly switch between display profiles
  using your Numpad keys. Each profile changes gamma, contrast,
  and digital vibrance / saturation - perfect for gaming, video
  editing, or quickly boosting visibility in dark scenes.

  The tool auto-detects your GPU (NVIDIA or AMD) and uses the
  correct driver API automatically. On systems without a
  supported GPU, gamma and contrast still work via Windows API.

  The app runs silently in your system tray and uses almost
  no resources. No installation required.

FEATURES
  - 3 switchable display profiles via Numpad 1/2/3
  - Adjusts Gamma, Contrast, and Digital Vibrance/Saturation
  - Supports NVIDIA (NvAPI) and AMD (ADL SDK) graphics cards
  - Auto-detects GPU - no configuration needed
  - Works with Shift held down (no conflicts)
  - Per-monitor control (choose one or all monitors)
  - Notifications can be turned on/off
  - English and German language support
  - Custom icon with profile number indicator
  - All settings saved automatically
  - Resets everything to normal on exit
  - Portable: no installation, just one EXE + icon file

PROFILES
  Num 1:  Normal    - Gamma 1.0, Contrast 50%, Vibrance 50%
  Num 2:  Bright    - Gamma 1.5, Contrast 55%, Vibrance 60%
  Num 3:  Brighter  - Gamma 2.0, Contrast 55%, Vibrance 70%

SYSTEM REQUIREMENTS
  - Windows 10 or 11
  - NVIDIA (GeForce, RTX, Quadro) or AMD (Radeon, RX) GPU
  - .NET Framework 4.0 (pre-installed on Windows 10/11)
  - Also works on Intel/other GPUs (Gamma + Contrast only)

FIRST LAUNCH - IMPORTANT
  On first launch, BrightRaider needs to make a small, safe
  change to your Windows registry. Here is exactly what
  happens and why:

  1. You double-click BrightRaider.exe
  2. Windows shows a UAC prompt ("Do you want to allow
     this app to make changes?") - click YES
  3. This creates ONE registry entry:

     Location: HKLM\SOFTWARE\Microsoft\Windows NT\
               CurrentVersion\ICM
     Name:     GdiIcmGammaRange
     Value:    256

     WHAT DOES THIS DO?
     Windows has a built-in restriction that blocks apps
     from adjusting display gamma. This registry entry
     simply tells Windows: "Allow gamma adjustments."
     It does NOT change any display settings itself.
     It does NOT affect system stability or security.
     It is used by many display calibration tools.
     It can be safely removed at any time.

  4. RESTART YOUR PC (only needed once!)
  5. Start BrightRaider.exe again - done!

  The UAC prompt only appears ONCE. After that, the app
  runs without any admin rights.

USAGE
  Hotkeys:
    Numpad 1  ->  Profile 1 (Normal)
    Numpad 2  ->  Profile 2 (Bright)
    Numpad 3  ->  Profile 3 (Brighter)

    Works with or without Shift, NumLock on or off.

  Tray Icon (right-click):
    - Switch profiles
    - Select monitor (one or all)
    - Toggle notifications
    - Switch language (English/Deutsch)
    - Exit

  The tray icon shows the app logo with a small profile
  number overlay:
    Green number  = Profile 1 (Normal)
    Orange number = Profile 2 (Bright)
    Red number    = Profile 3 (Brighter)

MONITOR SELECTION
  By default, all monitors change together. To change only
  one monitor: right-click the tray icon and select the
  specific display. Your choice is saved automatically.

NOTES
  - Numpad 1-3 are captured while the tool is running.
    Regular number keys (above letters) are NOT affected.
  - Digital Vibrance (NVIDIA) / Saturation (AMD) changes are
    visible in the respective GPU control panel.
    Gamma/Contrast changes are NOT shown there
    (this is normal - they work via a different API).
  - The tray menu shows your detected GPU at the bottom
    (NVIDIA, AMD, or None).
  - If something looks wrong: just exit the app. Everything
    resets to normal automatically.

UNINSTALL
  1. Exit BrightRaider (right-click tray -> Exit)
  2. Delete the folder with BrightRaider.exe, Icon.png,
     and BrightRaider.cfg
  3. Optional: Remove the registry entry:
     Open regedit, navigate to:
     HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\ICM
     Delete the value "GdiIcmGammaRange"
     (This is completely optional - it causes no harm
     if left in place)

[ DEUTSCH ]

WAS IST BRIGHTRAIDER?
  BrightRaider ist ein schlankes Tool fuer NVIDIA und AMD
  Grafikkarten, mit dem du per Tastendruck zwischen Display-
  Profilen wechseln kannst. Jedes Profil aendert Gamma,
  Kontrast und Digitale Farbanpassung/Saettigung gleichzeitig
  - perfekt fuer Gaming, Videobearbeitung oder um in dunklen
  Szenen mehr zu erkennen.

  Das Tool erkennt automatisch deine GPU (NVIDIA oder AMD)
  und nutzt die richtige Treiber-API. Auf Systemen ohne
  unterstuetzte GPU funktionieren Gamma und Kontrast trotzdem
  ueber die Windows-API.

  Die App laeuft leise im System-Tray und verbraucht kaum
  Ressourcen. Keine Installation noetig.

FUNKTIONEN
  - 3 umschaltbare Display-Profile ueber Numpad 1/2/3
  - Aendert Gamma, Kontrast und Digitale Farbanpassung/Saettigung
  - Unterstuetzt NVIDIA (NvAPI) und AMD (ADL SDK) Grafikkarten
  - Automatische GPU-Erkennung - keine Konfiguration noetig
  - Funktioniert auch mit gedrueckter Shift-Taste
  - Einzelne Monitore oder alle gleichzeitig steuerbar
  - Benachrichtigungen ein/ausschaltbar
  - Englisch und Deutsch
  - App-Icon mit Profilnummer-Anzeige
  - Alle Einstellungen werden automatisch gespeichert
  - Setzt beim Beenden alles auf Normal zurueck
  - Portabel: nur eine EXE + Icon-Datei, keine Installation

PROFILE
  Num 1:  Normal  - Gamma 1.0, Kontrast 50%, Farbanpassung 50%
  Num 2:  Hell    - Gamma 1.5, Kontrast 55%, Farbanpassung 60%
  Num 3:  Heller  - Gamma 2.0, Kontrast 55%, Farbanpassung 70%

SYSTEMVORAUSSETZUNGEN
  - Windows 10 oder 11
  - NVIDIA (GeForce, RTX, Quadro) oder AMD (Radeon, RX) GPU
  - .NET Framework 4.0 (auf Windows 10/11 vorinstalliert)
  - Funktioniert auch auf Intel/anderen GPUs (nur Gamma + Kontrast)

ERSTER START - WICHTIG
  Beim ersten Start muss BrightRaider eine kleine, sichere
  Aenderung in der Windows-Registry vornehmen. Hier ist genau
  erklaert was passiert und warum:

  1. Du doppelklickst BrightRaider.exe
  2. Windows zeigt eine UAC-Abfrage ("Moechten Sie zulassen,
     dass diese App Aenderungen vornimmt?") - klicke JA
  3. Es wird EIN Registry-Eintrag erstellt:

     Ort:   HKLM\SOFTWARE\Microsoft\Windows NT\
            CurrentVersion\ICM
     Name:  GdiIcmGammaRange
     Wert:  256

     WAS MACHT DIESER EINTRAG?
     Windows hat eine eingebaute Beschraenkung, die Apps
     daran hindert, die Display-Gamma-Werte zu aendern.
     Dieser Registry-Eintrag sagt Windows einfach:
     "Erlaube Gamma-Anpassungen."
     Er aendert KEINE Bildschirmeinstellungen selbst.
     Er beeinflusst NICHT die Systemstabilitaet.
     Er ist KEIN Sicherheitsrisiko.
     Er wird von vielen Kalibrierungs-Tools verwendet.
     Er kann jederzeit wieder entfernt werden.

  4. PC NEU STARTEN (nur einmalig noetig!)
  5. BrightRaider.exe erneut starten - fertig!

  Die UAC-Abfrage kommt nur EINMAL. Danach laeuft die App
  ohne Admin-Rechte.

BEDIENUNG
  Hotkeys:
    Numpad 1  ->  Profil 1 (Normal)
    Numpad 2  ->  Profil 2 (Hell)
    Numpad 3  ->  Profil 3 (Heller)

    Funktioniert mit und ohne Shift, NumLock an oder aus.

  Tray-Icon (Rechtsklick):
    - Profile wechseln
    - Monitor auswaehlen (einzeln oder alle)
    - Benachrichtigungen umschalten
    - Sprache wechseln (English/Deutsch)
    - Beenden

  Das Tray-Icon zeigt das App-Logo mit kleiner Profilnummer:
    Gruene Nummer  = Profil 1 (Normal)
    Orange Nummer  = Profil 2 (Hell)
    Rote Nummer    = Profil 3 (Heller)

MONITOR-AUSWAHL
  Standardmaessig wechseln alle Monitore gleichzeitig. Um nur
  einen Monitor zu aendern: Rechtsklick auf das Tray-Icon und
  den gewuenschten Monitor auswaehlen. Die Auswahl wird
  automatisch gespeichert.

HINWEISE
  - Numpad 1-3 sind belegt solange das Tool laeuft.
    Die normalen Zahlentasten (ueber den Buchstaben) sind
    NICHT betroffen.
  - Digitale Farbanpassung (NVIDIA) / Saettigung (AMD) wird
    im jeweiligen GPU Control Panel angezeigt.
    Gamma/Kontrast-Aenderungen werden dort NICHT angezeigt
    (das ist normal - sie nutzen eine andere API).
  - Im Tray-Menue wird unten die erkannte GPU angezeigt
    (NVIDIA, AMD oder Keiner).
  - Falls etwas komisch aussieht: einfach die App beenden.
    Alles wird automatisch auf Normal zurueckgesetzt.

DEINSTALLATION
  1. BrightRaider beenden (Rechtsklick Tray -> Beenden)
  2. Den Ordner mit BrightRaider.exe, Icon.png und
     BrightRaider.cfg loeschen
  3. Optional: Registry-Eintrag entfernen:
     regedit oeffnen, navigieren zu:
     HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\ICM
     Den Wert "GdiIcmGammaRange" loeschen
     (Komplett optional - der Eintrag ist harmlos und kann
     auch einfach stehen bleiben)
